# Offline Assets Gallery

این ماژول یک گالری آفلاین مبتنی بر دارایی‌ها را اجرا می‌کند. برای نمایش محصولات کافی است تصاویر و فایل شاخص را در مسیرهای زیر قرار دهید.

## قرار دادن تصاویر

تصاویر هر برند را در مسیر زیر کپی کنید:

```
app/src/main/assets/catalog/<brand>/<code>.jpg
```

فایل‌ها می‌توانند از هر نام دلخواهی پیروی کنند، کافی است در `index.json` به همان مسیر اشاره کنید.

## فایل `index.json`

فایل `app/src/main/assets/catalog/index.json` شامل آرایه‌ای از آیتم‌ها است که هر آیتم فیلدهای زیر را دارد:

- `id`: شناسه یکتا برای محصول
- `brand`: نام برند که برای نمایش در زیرنام استفاده می‌شود
- `name`: عنوان محصول که در کارت نمایش داده می‌شود
- `img`: مسیر نسبی تصویر داخل پوشهٔ دارایی‌ها (مثلاً `catalog/my/100001.jpg`)
- `price`: قیمت محصول (اختیاری)؛ در صورت وجود به‌صورت عددی خوانده و فرمت می‌شود

## بیلد APK دیباگ

برای ساخت APK دیباگ، از دستورات زیر استفاده کنید:

```bash
cd android_gallery_template
./gradlew app:assembleDebug
```

خروجی در مسیر زیر قرار می‌گیرد:

```
app/build/outputs/apk/debug/app-debug.apk
```
